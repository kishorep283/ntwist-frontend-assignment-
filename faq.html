JavaScript FAQ

What is JavaScript?

JavaScript is a scripting language used to make web pages interactive.

What is the difference between var, let, and const?

var is function scoped, let and const are block scoped. const cannot be reassigned.

What is a function?

A function is a block of code that performs a specific task and can be reused.

What is an array in JavaScript?

An array is a collection of items stored in a single variable.

What is an object in JavaScript?

An object is a collection of key-value pairs used to store multiple values.
